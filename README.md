# Spritesheet Generator

## English

Spritesheet generator is a tool to create spritsheet from a folder of images.

## Français

Spritesheet Générator est un outil de création de spritesheet à partir d'un dossier d'images.

Le script va explorer le dossier et ajouter chaques images sur une grille pour former un seul fichier contenant toutes les animations.

### Installation

- Clonez ou télécharger les sources
- Installer Python sur votre machine
- Installer pipenv `py -m pip install --user pipenv `
- Lancer le programme `py -m pipenv run python main.py`
